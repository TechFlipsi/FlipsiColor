## Key Concepts Summary

- **Zero Setup**: Install → start → use. No extra software, no manual model downloads.
- **Local First**: Everything runs locally. No cloud, no account, no subscription.
- **Non-Destructive**: Original files are NEVER modified.
- **3 Modes**: Ask (👍👎), Smart-Learn (silent), Turbo (zero interaction)
- **3 Intensity Levels**: Leicht (minimal), Mittel (professional, default), Stark (AI takes over)
- **2-Round Learning**: R1: 0-60 feedbacks (basics), R2: 61-120 (variety), Autonomous: 120+
- **Auto-Retrain**: New scene type → mini 5-10 feedback cycle
- **200MB Knowledge Base limit** with 3-stage auto-compaction
- **Camera-Universal**: Lensfun 500+ profiles
- **GPU Required**: CUDA / DirectML / Metal
