## Architecture

- **Source**: `.ts` XML files in `i18n/` (German as source language)
- **Compiled**: `.qm` binary files generated by `Qt6::lrelease`
- **Load order**: User setting → System locale → German fallback
- **Code identifiers**: German method names (`gpuVerfuegbar()`, `Intensitaet::Leicht`)

## Adding a New Language

1. Create `i18n/flipsicolor_XX.ts`
2. Add `XX` to `FLIPSICOLOR_LANGUAGES` in `CMakeLists.txt`
3. Add entry to `VERFUEGBARE_SPRACHEN` and `SPRACHNAMEN` in `main.cpp`
4. Fill in translations
5. Build → `.qm` auto-generated

See [LANGUAGE.md](LANGUAGE.md) for full documentation (German).
